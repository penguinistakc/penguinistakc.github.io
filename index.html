Summary: Latitude vs. Weather


Visualizations



The purpose of this project was to analyze how weather changes as one gets closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap, using 1,779 samples to assemble the base dataset.

After assembling the dataset, I used the matplotlib library to plot various aspects of the weather vs latitude. Factors considered included temperature, cloud cover, wind velocity and humidity.

This site provides source data and visualizations as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
